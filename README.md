# points-list
# Installation
1.Please download the zip file to your local machine and enter the directory

2.Ensure Python runtime is installed
```cmd
pip install -r requirements.txt
```
3.Run the apps main program
```web
your_ip:80
```
4.It is recommended to change the password. Run change_password.py to modify it
# Frontend Modifications
All modifiable content is located in config.json

1.page_title is the page title

2.title is the title within the page

3.content is the main body content

4.footer is the footer content
# Backup
Please backup the following files:

config.json Frontend custom content

key.json Password hash value

students_data.json Score data
# Additional Terms
